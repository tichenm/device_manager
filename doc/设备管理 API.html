1.1新增设备

基本信息

Path： /system/screen
Method： POST

接口描述

创建一个门禁

请求参数
名称 类型 是否必须 备注 示例 其他信息
camera_address String 是 摄像头地址
camera_type String 是 摄像头类型
camera_position String 是 摄像头位置
返回值说明：

| 字段 | 类型 | 说明 |
|--------|--------|
| camera_address | String | 摄像头地址 |
|camera_type|String|摄像头类型|
| camera_position | String | 摄像头位置 ||

返回JSON：
    {
      "code": 0,
      "data": {
      "id":1,
      "camera_address":"192.68.18.121",
      "camera_position":"xxxxx",
      "camera_type":"巨龙",
      }
    }

2.1新增账号

基本信息

Path： /system/account
Method： POST

接口描述

创建一个使用账户

请求参数
名称 类型 是否必须 备注 示例 其他信息
ip_address String 是 IP地址
account String 是 用户名
password String 是 密码
activation int {在使用：1，未使用：0} 默认0
返回值说明：

| 字段 | 类型 | 说明 |
|--------|--------|
| ip_address | String | IP地址 |
| account | String | 用户名 |
| password | String | 密码 ||

返回JSON：
    {
      "code": 0,
      "data": {
      "id":1,
      "ip_address":"192.68.18.121",
      "account":"xxxxx",
      "password":"xxxxx"
      "activation":0
      }
    }
